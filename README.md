# QuizApp

This project is a quiz app I developed in highschool using Flutter. It uses an API to submit login information to the database. The API is titled main.py and can be found in this repository.

#Getting Started
All the files involved with the main code will be under the lib folder, including main.dart and other custom classes I built for this project. I recommend running the API in a Pycharm window and running the rest of the project in Android Studio with Flutter.
